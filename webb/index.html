När kom WWW och vem skapade det

. WWW upstodd 1989 och kunde användas av alla. Det var Tim Berners-lee och Robert Cailliau som skapade sidan. WWW är en förkortning för World Wide Webb,

Vad var teknologin bakom HTTP WWW och DNS

. DNS är en databas somöversätter domännnamn till IP adresser som en telefonkatalog som översätter telefonnummer till namn DNS gör så att det går snabbt att hitta relevant information

. HTTP är ett kommunikationsprotokoll som används på internet för att överföra webbsidor mellan webbserver, den skickar information mellan wbbplats of webbläsare

. WWW är ett sammankopplat nätverk av webbplatser, dokument som kan nås via internet det är ett informations system där information kan nåst via URL
